# Sidebar-Project-and-Modal-Project-using-React.js 


The Project is about a sidebar and a modal, you will see the the full navbar on a bigger screen but once you view it on a smaller screen you will see the sidebar with a toggle icon to close and open the side bar. There is also a show modal button at the center of the page, once you click on that button the modal content will pop-up, and you can also close the modal with a X Button.
